Практичне застосування елементів CSS

На цій сторінці буде розписано які стилі використовувались
Цей стиль використовувався для задання body кольору фону, шрифту і кольору тексту та задання відступів
body {
background-color: #eaeaea;
font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
color: #333;
margin: 0;
}
Цей стиль використовувався для задання блоку header кольору фону, кольору тексту та задання відступів
header { background-color: #3498db;
color: white;
padding: 20px;
}
Наступні два стилі задають оформлення для заголовків h1 та h2

Приклад заголовку h1


Приклад заголовку h2


h1
{
text-align: center;
font-family: Impact, fantasy;
}
h2
{
text-align: center;
font-family: Comic Sans MS, Comic Sans, cursive;
}
Цей стиль офорилює навігаційну панель, задає їй колір фону та відступи
nav {
background-color: #5dade2;
padding: 10px;
}
Цей стиль задає стиль посилань у блокі навігації, та задає колір шрифту і відступи
nav a {
color: white;
margin-right: 15px;
}
Стиль для блоку section задає відступи та колір фону
section {
padding: 20px;
background-color: #B0E0E6;
}
Стиль для тегу футер використовується для створення фіксованого, блоку внизу сторінки та задає колір фону та тексту сторінки
footer {
background-color: #3498db;
color: white;
text-align: center;
padding: 10px;
position: fixed;
bottom: 0;
width: 100%;
}
